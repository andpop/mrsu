# Task02 — Игра "Крестики-нолики"

## Описание проекта
Программа для консольной игры **Крестики-нолики** (tic-tac-toe) с компьютером. Размер поля можно выбрать от 3x3 до 10x10.

---

### Правила игры
- Всегда начинают крестики.  
- Кем играет человек — определяется случайным образом. 
- Игрок и компьютер по очереди делают ходы, пока не появится победитель или не заполнится всё поле.
- После окончания игры результат сохраняется в базе данных SQLite.  
- Для каждой партии сохраняется:
  - размер поля;
  - дата игры;
  - имя игрока;
  - фигура, которой играет человек (крестики или нолики);
  - победитель;
  - запись всех ходов (номер хода | координата X | координата O). 
- Игра продолжается, пока число не будет угадано.  

---

### Режимы работы программы
1. Начать новую игру.  
2. Список всех сохранённых партий. 
3. Повтор любой сохранённой партии (показ всех ходов). 
4. Выход.  

#### Поддерживаются следующие параметры командной строки:
- --new, -n — начать новую игру (режим по умолчанию).

- --list, -l — список всех сохранённых партий (не реализовано, выводится сообщение).

- --replay <id>, -r <id> — повтор сохранённой партии (не реализовано, выводится сообщение).

- --help, -h — показать справку по командам.

#### Параметры запуска
Новая игра:
```bash
php bin/tic-tac-toe --new
```

или короче:
```bash
tic-tac-toe -n
```

Показать справку:
```bash
php bin/tic-tac-toe --help
```

Попробовать вывести список игр:
```bash
php bin/tic-tac-toe --list
```

#### Формат ввода координат
Ход указывается в виде двух чисел:
```bash
строка столбец
```
Например, чтобы походить в центр на поле 3x3, нужно ввести:
```bash
2 2
```

Строки и столбцы нумеруются с 1.
---


## Требования
- **PHP** >= 8.0 (CLI-режим)  
- **Composer** (глобально установленный)  
- **SQLite** (для хранения данных игры)  

---

# Игра "Холодно-горячо" (Slim Framework)

Single Page Application с PHP backend на микрофреймворке Slim и SQLite базой данных.

## 🚀 Запуск приложения

1. Перейдите в каталог Task09:
   ```bash
   cd Task09
   ```

2. Установите зависимости через Composer:
   ```bash
   composer install
   ```

3. Запустите встроенный PHP-сервер:
   ```bash
   php -S localhost:3000 -t public
   ```

4. Откройте в браузере:
   ```
   http://localhost:3000
   ```
   или
   ```
   http://localhost:3000/index.html
   ```

## 📁 Структура проекта

```
Task09/
├── public/              # Корень веб-сервера

│   ├── css/
│   │   └── style.css    # Стили
│   └── js/
│       └── app.js       # JavaScript (SPA логика)
├── src/
│   └── Database.php     # Работа с SQLite
├── db/
│   └── cold-hot.db      # База данных (создаётся автоматически)
├── vendor/              # Зависимости Composer
├── composer.json
├── composer.lock
└── README.md
```

## 🔌 REST API (Slim Framework)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | / | Редирект на /index.html |
| GET | /index.html | HTML страница игры |
| GET | /games | Список всех игр |
| GET | /games/{id} | Данные игры с ходами |
| POST | /games | Создать новую игру |
| POST | /step/{id} | Сохранить ход |

## 📋 Правила игры

1. Компьютер загадывает трёхзначное число без повторяющихся цифр
2. Первая цифра не может быть нулём
3. После каждой попытки вы получаете подсказки:
   - 🔥 **Горячо** — цифра на своём месте
   - 🌡️ **Тепло** — цифра есть, но не на своём месте
   - ❄️ **Холодно** — такой цифры нет в числе

## 🛠 Технологии

- Frontend: HTML5, CSS3, JavaScript (ES6)
- Backend: PHP 8.0+ с Slim Framework 4
- База данных: SQLite
- Архитектура: SPA + REST API

## 📦 Зависимости

- slim/slim ^4.0
- slim/psr7 ^1.0
- slim/http ^1.0

# Игра "Холодно-горячо" (Web-версия с IndexedDB)

Браузерная версия игры, в которой нужно угадать трёхзначное число без повторяющихся цифр.

## 🎮 Демо

**Играть онлайн:** [https://relflly.github.io/cold-hot-web/](https://relflly.github.io/cold-hot-web/)

## 📋 Правила игры

1. Компьютер загадывает трёхзначное число без повторяющихся цифр
2. Первая цифра не может быть нулём
3. После каждой попытки вы получаете подсказки:
   - 🔥 **Горячо** — цифра на своём месте
   - 🌡️ **Тепло** — цифра есть в числе, но не на своём месте
   - ❄️ **Холодно** — такой цифры нет в числе
4. Подсказки выводятся в алфавитном порядке, чтобы не раскрывать позиции цифр

## 💾 Работа с базой данных

Игра использует **IndexedDB** для хранения данных в браузере:

- **Сохранение партий** — каждая игра автоматически сохраняется
- **История попыток** — все ходы записываются в базу данных
- **Просмотр истории** — вкладка "История" показывает все сыгранные партии
- **Воспроизведение** — можно посмотреть ход любой партии
- **Удаление** — ненужные партии можно удалить

### Структура базы данных

- **games** — хранилище игровых сессий (id, player_name, secret_number, outcome, created_at)
- **attempts** — хранилище попыток (id, game_id, attempt_number, guess, hints)

## 🚀 Запуск локально

Просто откройте файл `index.html` в браузере.

Или запустите локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve
```

Затем откройте http://localhost:8000

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Gradients)
- JavaScript ES6 (Modules, async/await)
- IndexedDB (через библиотеку idb)

# Игра "Холодно-горячо" (cold-hot)

Консольная игра, в которой нужно угадать трехзначное число без повторяющихся цифр.

## Требования

- PHP 8.0 или выше
- Composer
- Расширения: ext-pdo, ext-pdo_sqlite

## Установка

```bash
composer install
```

## Запуск

```bash
# Начать новую игру
./bin/cold-hot --new
./bin/cold-hot -n

# Показать список сохраненных партий
./bin/cold-hot --list
./bin/cold-hot -l

# Воспроизвести партию по ID
./bin/cold-hot --replay 1
./bin/cold-hot -r 1

# Показать справку
./bin/cold-hot --help
./bin/cold-hot -h
```

## Правила игры

1. Компьютер загадывает трехзначное число без повторяющихся цифр
2. Первая цифра не может быть нулем
3. После каждой попытки выводятся подсказки:
   - **Горячо** - цифра на своем месте
   - **Тепло** - цифра есть в числе, но не на своем месте
   - **Холодно** - такой цифры нет в числе
4. Подсказки выводятся в алфавитном порядке

## Особенности версии Task05

В этой версии работа с базой данных реализована через ORM RedBeanPHP вместо прямых SQL-запросов.

# Игра "Холодно-горячо" (Cold-Hot)

Консольная игра, в которой нужно угадать трехзначное число без повторяющихся цифр.

## Установка

### Через Composer (глобально)

```bash
composer global require ganchenkov/cold-hot
```

### Через Composer (локально)

```bash
composer require ganchenkov/cold-hot
```

### Из исходников

```bash
git clone https://github.com/Relflly/cold-hot.git
cd cold-hot
composer install
```

## Запуск игры

```bash
# После глобальной установки
cold-hot

# После локальной установки
./vendor/bin/cold-hot

# Из исходников
php bin/cold-hot
```

## Параметры командной строки

| Параметр | Краткий | Описание |
|----------|---------|----------|
| `--new` | `-n` | Начать новую игру (по умолчанию) |
| `--list` | `-l` | Показать список сохраненных партий |
| `--replay <ID>` | `-r <ID>` | Воспроизвести партию по ID |
| `--help` | `-h` | Показать справку |

## Правила игры

1. Компьютер загадывает трехзначное число без повторяющихся цифр
2. Первая цифра не может быть нулем (число от 102 до 987)
3. Игрок вводит свои предположения в формате трех цифр (например: `123`, `456`, `789`)
4. После каждой попытки выводятся подсказки:
   - **Горячо** — цифра угадана и находится на своем месте
   - **Тепло** — цифра есть в числе, но не на своем месте
   - **Холодно** — такой цифры нет в загаданном числе
5. Подсказки выводятся в алфавитном порядке

## Пример игры

```
===========================================
   Добро пожаловать в игру "Холодно-горячо"!
===========================================

Введите ваше имя: Игрок
Введите трехзначное число: 123
Подсказки: Тепло Холодно Холодно
Введите трехзначное число: 456
Подсказки: Горячо Тепло Холодно
Введите трехзначное число: 465

Поздравляем! Вы угадали число за 3 попыток!
```

## Работа с базой данных

Игра автоматически сохраняет все партии в базу данных SQLite.

### Сохраняемая информация

Для каждой игры сохраняется:
- Дата и время игры
- Имя игрока
- Загаданное компьютером число
- Результат игры (угадал/не угадал)
- Все попытки с подсказками

### Просмотр сохраненных партий

```bash
cold-hot --list
```

Пример вывода:
```
=== Список сохраненных партий ===

ID   | Дата                 | Игрок           | Число  | Результат
-----------------------------------------------------------------
3    | 2025-12-21 23:45:00  | Игрок           | 465    | угадал
2    | 2025-12-21 22:30:00  | Тест            | 123    | угадал
1    | 2025-12-21 21:15:00  | Новичок         | 789    | угадал
```

### Воспроизведение партии

```bash
cold-hot --replay 3
```

Пример вывода:
```
=== Воспроизведение партии #3 ===

Игрок: Игрок
Дата: 2025-12-21 23:45:00
Загаданное число: 465
Результат: угадал

Попытки:
  1. 123 -> Тепло Холодно Холодно
  2. 456 -> Горячо Тепло Холодно
  3. 465 -> Горячо Горячо Горячо
```

### Расположение базы данных

База данных хранится в файле:
- Windows: `%USERPROFILE%\.cold-hot\cold-hot.db`
- Linux/Mac: `~/.cold-hot/cold-hot.db`

## Формат ввода

- Вводите ровно 3 цифры без пробелов и других символов
- Примеры корректного ввода: `123`, `456`, `789`, `102`
- Примеры некорректного ввода: `12`, `1234`, `abc`, `1 2 3`

## Требования

- PHP >= 8.0
- ext-pdo
- ext-pdo_sqlite
- Composer

# Игра "Холодно-Горячо"

Программа для консольной игры **"Холодно-Горячо"**.  
Игрок пытается угадать число, загаданное компьютером, получая текстовые подсказки о том, насколько близок его вариант к правильному.

---
### Правила игры

- Игрок пытается угадать трехзначное число с уникальными цифрами.
- После каждой попытки компьютер выдает подсказки трех типов:
    - "Холодно" - ни одна цифра не отгадана
    - "Тепло" - одна цифра отгадана, но не отгадана ее позиция
    - "Горячо" - одна цифра и ее позиция отгадана
- Подсказки сортируются в алфавитном порядке
- Максимальное количество попыток: 10

## Игровой процесс

1. При запуске отображается главное меню
2. Игрок вводит номер пункта меню и нажимает **Enter**.
3. После старта игры программа сообщает что число 3-х значное число было загадано.
4. Игрок на каждой попытке вводит целое число в указанном диапазоне и нажимает **Enter**.
5. После каждой попытки программа выводит подсказку:
   - `Горячо` — одна из цифр находится на правильном месте
   - `Тепло` — одна из цифр правильна, но находится не на своей позиции
   - `Холодно` — одно из цифр не входит в число
6. Игра продолжается, пока число не будет угадано за 10 попыток.
7. Все данные игры (дата, имя игрока, максимальное число, загаданное число, результат, список всех попыток с подсказками) автоматически сохраняются в локальную базу **SQLite**.

---

## Формат ввода

- При угадывании числа: **целое число** из диапазона от `100` до выбранного максимального значения.  
- Никаких специальных символов или координат поля не требуется — вводятся только числа.

---

## Режимы работы программы
1. **Начать новую игру** — запустить игру с выбранными параметрами.  
2. **Логи игр** — просмотр записей всех игр.  
3. **Просмотр игры по ID** — просмотр записи игры по ID.  
4. **Правила игры** — справка о правилах и подсказках.  
5. **Выход** — завершение программы.

---

## Требования

- **PHP** ≥ 8.0 (CLI-режим)  
- **Composer** (установлен глобально)  
- **SQLite** (для хранения статистики игр)

---

## Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Baklaniso/cold-hot.git
cd cold-hot

# 2. Установить зависимости
composer install

# 3. Сгенерировать автозагрузку
composer dump-autoload -o

# 4. Запустить игру
./bin/cold-hot
```

# Игра "Холодно-Горячо"

Программа для консольной игры **"Холодно-Горячо"**.  
Игрок пытается угадать число, загаданное компьютером, получая текстовые подсказки о том, насколько близок его вариант к правильному.

---
### Правила игры

- Игрок пытается угадать трехзначное число с уникальными цифрами.
- После каждой попытки компьютер выдает подсказки трех типов:
    - "Холодно" - ни одна цифра не отгадана
    - "Тепло" - одна цифра отгадана, но не отгадана ее позиция
    - "Горячо" - одна цифра и ее позиция отгадана
- Подсказки сортируются в алфавитном порядке
- Максимальное количество попыток: 10

## Игровой процесс

1. При запуске отображается главное меню
2. Игрок вводит номер пункта меню и нажимает **Enter**.
3. После старта игры программа сообщает что число 3-х значное число было загадано.
4. Игрок на каждой попытке вводит целое число в указанном диапазоне и нажимает **Enter**.
5. После каждой попытки программа выводит подсказку:
   - `Горячо` — одна из цифр находится на правильном месте
   - `Тепло` — одна из цифр правильна, но находится не на своей позиции
   - `Холодно` — одно из цифр не входит в число
6. Игра продолжается, пока число не будет угадано за 10 попыток.
7. Все данные игры (дата, имя игрока, максимальное число, загаданное число, результат, список всех попыток с подсказками) автоматически сохраняются в локальную базу **SQLite**.

---

## Формат ввода

- В главном меню: введите **цифру от 1 до 5** и нажмите Enter.  
- При угадывании числа: **целое число** из диапазона от `100` до выбранного максимального значения.  
- Никаких специальных символов или координат поля не требуется — вводятся только числа.

---

## Режимы работы программы
1. **Начать новую игру** — запустить игру с выбранными параметрами.  
2. **Таблица рекордов** — просмотр сохранённых результатов прошлых игр.  
3. **Настройки сложности** — выбор максимального числа для новых игр.  
4. **Правила игры** — справка о правилах и подсказках.  
5. **Выход** — завершение программы.

---

## Требования

- **PHP** ≥ 8.0 (CLI-режим)  
- **Composer** (установлен глобально)  
- **SQLite** (для хранения статистики игр)

# Игра "Холодно-Горячо"

Программа для консольной игры **"Холодно-Горячо"**.  
Игрок пытается угадать число, загаданное компьютером, компьютер загадывает **трёхзначное число без повторяющихся цифр**, а игрок пытается его угадать.

## [Играть](http://localhost:3000/)

## Как играть (Slim)

1. **Клонируйте или скачайте проект**
2. В каталоге Task09:
   1) Установить зависимости:
      composer install
   2) Обновить автозагрузку:
      composer dump-autoload -o
   3) Запустить встроенный сервер:
      php -S localhost:3000 -t public public/index.php
   4) Открыть: http://localhost:3000/
   5) REST:
   - GET  /games
   - GET  /games/{id}
   - POST /games       { "player_name": "...", "secret": "123" }
   - POST /step/{id}   { "turn": 1, "guess": "123", "hints": ["..."], "outcome": "win|lose|in_progress" }

3. Введите имя игрока и нажмите «Начать».
4. Вводите трёхзначные числа без повторяющихся цифр (например, `123`).  
   Некорректный ввод не засчитывается.
5. Команда `exit` — досрочный выход (покажем секрет).
6. После окончания партия сохраняется в **IndexedDB**. Раздел «История» позволяет:
   - просмотреть список партий,
   - повторить любую (пошаговые ходы и подсказки),
   - очистить локальную БД.

### Работа с базой (IndexedDB)
– Список партий: раздел «История».  
– Повтор партии: нажмите «Повтор» в таблице или введите ID и нажмите «Повтор по ID».  
– Прямая ссылка: `#replay-<ID>` (например, `#replay-12`).  
– Очистить локальную БД: кнопка «Очистить БД» в разделе «История».

## Технологии
- HTML5, CSS3, JavaScript, php
- SQLite

## Требуются расширения PHP
- pdo_sqlite
- sqlite3
